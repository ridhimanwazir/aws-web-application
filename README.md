# Fullstack-Application
Deploying full stack application( Task Manager) on AWS using AWS Using Amplify, Elastic Beanstalk, and EKS

### The application allows users to list, create, and delete tasks, which are stored in dynamodb. The user gets emails whenever a new task is created.

## Steps

1. Create react frontend application and deploy on aws amplify
2. Create Python middleware to serve API endpoints that interact with dynamodb and sns topic.
3. Set up the EKS cluster and required environment to deploy the backend service to send emails using sns when ever a new task is added.
